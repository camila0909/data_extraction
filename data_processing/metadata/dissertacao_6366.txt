Abstract

In this work we study Gr¨obner basis to present an answer for the Three Color Problem

that consist to decide if a given map can or not to paint using only three colors and, if is

the case, how to paint it. The main idea is translate the geometric problem to an algebraic

problem and for this we study system of polynomials equations, the more known application

of Gr¨obner basis, concepts and properties of polynomial rings in several variables and ideals

as well.

Key words: System of polynomials equations, Gr¨obner Basis, Three Color Problem.
