Abstract

This work aims to construct the usual numerical systems, namely the set of natural
numbers (N), the set of integers (Z), the set of rational numbers (Q) and the set of real
numbers (R). We begin the work dealing with notions about sets and binary relations.
Next, we present the set of natural numbers, de(cid:28)ned by Peano’s axioms; the set of

of rational numbers; end for all these cases we show the immersion of the previous set
in the ensemble that appears in the sequence. Finally, we observed some materials in
elementary school and high school in order to investigate how these themes are being
presented to the students.
